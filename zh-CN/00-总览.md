> 本参考手册由6个子智能体并行分析 1332 个 TypeScript 源文件后自动生成。
> 目的：作为设计 Agent 工作流和工具系统时的可复用模式库。

## 仓库结构

```
src/
├── Tool.ts                  # 工具基类接口 & buildTool 辅助函数
├── Task.ts                  # 任务基类抽象
├── QueryEngine.ts           # 主对话/查询引擎
├── commands.ts              # 斜杠命令注册中心
├── history.ts               # 对话历史管理
│
├── tools/                   # 30+ 工具实现
│   ├── AgentTool/           # 多智能体系统（核心）
│   ├── BashTool/            # Shell 执行（含安全过滤）
│   ├── FileEditTool/        # 文件编辑（搜索替换）
│   ├── FileReadTool/        # 文件读取（含限制）
│   ├── FileWriteTool/       # 文件创建/覆盖
│   ├── GlobTool/            # 文件模式匹配
│   ├── GrepTool/            # 内容搜索（ripgrep）
│   ├── WebFetchTool/        # URL 抓取 & HTML→MD
│   ├── WebSearchTool/       # 网页搜索
│   ├── MCPTool/             # Model Context Protocol
│   ├── SendMessageTool/     # 智能体间通信
│   ├── TaskCreateTool/      # 后台任务创建
│   ├── TodoWriteTool/       # 进度追踪
│   ├── SkillTool/           # 斜杠命令分发器
│   ├── EnterPlanModeTool/   # 计划模式入口
│   ├── ScheduleCronTool/    # 定时调度
│   ├── TeamCreateTool/      # Swarm 团队管理
│   └── shared/              # 共享工具函数
│
├── tasks/                   # 任务生命周期管理
│   ├── DreamTask/           # 后台"梦境"任务
│   ├── LocalShellTask/      # Shell 进程任务
│   ├── LocalAgentTask/      # 本地子智能体任务
│   ├── RemoteAgentTask/     # 远程智能体任务
│   └── InProcessTeammateTask/ # 进程内 Swarm 成员
│
├── coordinator/             # 编排器/领导者模式
├── bridge/                  # 远程连接（claude.ai/code）
├── server/                  # 直连 / SDK 服务器
├── ink/                     # 终端 UI 框架（React-for-CLI）
├── vim/                     # Vim 模式模拟
├── cli/                     # CLI 基础设施 & 传输层
├── commands/                # 斜杠命令实现
├── plugins/                 # 插件架构
├── utils/                   # 工具模块
│   ├── settings/            # 设置管理
│   ├── permissions/         # 权限系统
│   ├── memory/              # 记忆系统
│   ├── mcp/                 # MCP 工具函数
│   ├── shell/               # Shell 执行
│   ├── hooks/               # Hook 系统
│   ├── swarm/               # 多智能体协调
│   ├── sandbox/             # 沙箱
│   └── ...                  # 20+ 其他工具模块
├── memdir/                  # 智能体共享文件系统
├── bootstrap/               # 应用初始化
├── constants/               # 常量定义
├── types/                   # 类型定义
├── keybindings/             # 快捷键系统
├── buddy/                   # 伴侣/宠物功能
└── migrations/              # 设置迁移脚本
```
