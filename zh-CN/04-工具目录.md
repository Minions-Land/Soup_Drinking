## 文件系统工具

### FileReadTool（文件读取）
- **用途**：读取文件内容，带行号
- **特性**：二进制/PDF 检测、Token 限制、图像支持（多模态）
- **参数**：`file_path`、`offset?`、`limit?`、`pages?`（PDF）

### FileEditTool（文件编辑）
- **用途**：搜索替换式文本编辑
- **特性**：精确字符串匹配、`replace_all` 标志、LSP 诊断追踪
- **参数**：`file_path`、`old_string`、`new_string`、`replace_all?`

### FileWriteTool（文件写入）
- **用途**：创建或覆盖文件
- **参数**：`file_path`、`content`
- **约束**：覆盖前必须先读取文件

### GlobTool（文件模式匹配）
- **用途**：快速文件模式匹配（适用任何规模代码库）
- **参数**：`pattern`、`path?`
- **特性**：按修改时间排序，排除 VCS 目录

### GrepTool（内容搜索）
- **用途**：基于 ripgrep 的内容搜索
- **参数**：`pattern`、`path?`、`glob?`、`type?`、`output_mode`、`head_limit`、`offset`、`multiline?`
- **特性**：正则表达式、分页、文件类型过滤

### NotebookEditTool（Notebook 编辑）
- **用途**：编辑 Jupyter Notebook 单元格
- **参数**：`notebook_path`、`new_source`、`cell_id?`、`cell_type?`、`edit_mode`

## 执行工具

### BashTool（Shell 执行）
- **用途**：执行 Shell 命令
- **特性**：安全过滤（`bashSecurity.ts`）、路径验证、破坏性命令检测、沙箱支持
- **参数**：`command`、`description?`、`timeout?`、`run_in_background?`
- **子模块**：`commandSemantics.ts`（读/搜索/列表分类）、`sedValidation.ts`、`pathValidation.ts`
- **模式**：复杂的正则命令过滤，自动沙箱化

### PowerShellTool（PowerShell 执行）
- **用途**：执行 PowerShell 命令（Windows）

## 网络工具

### WebFetchTool（URL 抓取）
- **用途**：抓取 URL 内容，HTML 转 Markdown
- **特性**：自动 HTTP→HTTPS 升级、重定向处理、15 分钟缓存
- **参数**：`url`、`prompt`

### WebSearchTool（网络搜索）
- **用途**：网络搜索并格式化结果
- **参数**：`query`、`allowed_domains?`、`blocked_domains?`

## 智能体与通信工具

### AgentTool（智能体工具）
- **用途**：生成和管理子智能体
- **参数**：`description`、`prompt`、`subagent_type?`、`model?`、`isolation?`、`run_in_background?`
- **关键文件**：`runAgent.ts`、`forkSubagent.ts`、`builtInAgents.ts`、`loadAgentsDir.ts`
- **模式**：支持 Fork（上下文继承）、后台、工作树隔离

### SendMessageTool（消息发送）
- **用途**：向已有智能体发送消息
- **参数**：`to`（智能体 ID 或名称）、`message`
- **模式**：恢复智能体时保留完整上下文

### TeamCreateTool / TeamDeleteTool（团队管理）
- **用途**：创建/解散智能体团队

## 任务管理工具

### TaskCreateTool（创建任务）
- **用途**：创建后台任务

### TaskGetTool / TaskOutputTool（获取任务输出）
- **参数**：`task_id`、`block`（是否阻塞）、`timeout`

### TaskStopTool（停止任务）
- **参数**：`task_id`

### TaskListTool（列出任务）
- **用途**：Swarm 模式下的共享工作队列

## 规划与组织工具

### EnterPlanModeTool / ExitPlanModeTool（计划模式）
- **用途**：切换到计划模式处理复杂任务
- **模式**：限制为只读工具，然后呈现计划供用户批准

### TodoWriteTool（待办事项）
- **用途**：结构化任务列表管理
- **参数**：`todos[]`，含 `content`、`status`、`activeForm`

## MCP 工具

### MCPTool（MCP 工具）
- **用途**：执行 MCP 服务器工具
- **特性**：根据连接的 MCP 服务器 Schema 动态生成

### McpAuthTool（MCP 认证）
- **用途**：处理 MCP 服务的 OAuth 流程

### ReadMcpResourceTool / ListMcpResourcesTool（MCP 资源）
- **用途**：访问 MCP 服务器资源

- **模式**：偏移分钟调度避免集群热点
