# 工具函数与核心基础设施

> 🗺️ **导航**: [← README](../README.md) | [英文版](../en/08-UTILS-CORE.md)

> 源码位置：`src/utils/`、`src/context/`、`src/bootstrap/`、`src/constants/`、`src/types/`、`src/memdir/`

## 1. 核心引擎

### QueryEngine（`src/QueryEngine.ts`）
主对话循环：
- 发送对话历史到模型
- 处理工具调用
- 管理工具并发（并行 vs 串行）
- 应用工具结果预算
- 上下文过大时执行压缩

### 核心类型（`src/types/`）
- `message.ts` — UserMessage、AssistantMessage、SystemMessage、ProgressMessage
- `permissions.ts` — PermissionMode、PermissionResult、ToolPermissionRulesBySource
- `tools.ts` — 工具进度类型（BashProgress、AgentToolProgress 等）
- `hooks.ts` — Hook 类型（PreToolUse、PostToolUse 等）
- `ids.ts` — AgentId、SessionId 类型定义

## 2. 设置管理（`src/utils/settings/`）

| 文件 | 用途 |
|------|------|
| `settings.ts` | 核心设置读写 |
| `validation.ts` | Schema 验证 |
| `settingsCache.ts` | 性能缓存 |
| `applySettingsChange.ts` | 原子设置更新 |
| `changeDetector.ts` | 设置变更监控 |
| `mdm/` | 托管设备管理（企业） |
| `permissionValidation.ts` | 权限规则验证 |

## 3. 权限系统（`src/utils/permissions/`）

工具执行的安全网关：
- 实验性工具的特性门控
- 用户同意强制执行
- `denialTracking.ts` — 追踪重复拒绝，升级为询问
- 与 QueryEngine 集成应用安全过滤器

## 4. 记忆系统（`src/utils/memory/`）

| 文件 | 用途 |
|------|------|
| `versions.ts` | 记忆格式版本控制 |
| `types.ts` | 记忆条目类型 |

记忆以带 Frontmatter 的 Markdown 文件形式存储在 `.claude/projects/` 中。

## 5. MCP 集成（`src/utils/mcp/`）

Model Context Protocol 工具函数：
- 客户端连接管理
- 服务器注册
- 工具 Schema 转换（MCP → Claude 工具格式）

## 6. Shell 与进程（`src/utils/shell/`、`src/utils/bash/`）

Shell 执行基础设施：
- 进程生成和管理
- 环境变量处理
- 平台特定适配（macOS、Linux、Windows）

## 7. Hook 系统（`src/utils/hooks/`）

事件驱动的可扩展性：
- `PreToolUse` — 工具执行前
- `PostToolUse` — 工具执行后
- `session_start` — 会话初始化
- `pre_compact` / `post_compact` — 上下文压缩

## 8. 插件系统（`src/utils/plugins/`）

插件加载和生命周期：
- 多来源发现
- 激活/停用
- 能力注册（技能、Hook、MCP 服务器）

## 9. Swarm 协调（`src/utils/swarm/`）

| 文件 | 用途 |
|------|------|
| `spawnUtils.ts` | 启动队友 |
| `inProcessRunner.ts` | 进程内智能体的保活循环 |
| `permissionSync.ts` | 领导者 → 工作者权限传播 |
| `teammateModel.ts` | 队友模型选择 |
| `reconnection.ts` | 断线恢复 |
| `backends/TmuxBackend.ts` | 在 tmux 面板中生成 |
| `backends/ITermBackend.ts` | 在 iTerm 标签中生成 |
| `backends/InProcessBackend.ts` | 在同一 Node 进程中生成 |

## 10. 上下文管理（`src/context/`）

- 系统提示词组装
- 上下文窗口追踪
- 记忆文件注入
- CLAUDE.md 加载

## 11. 启动引导（`src/bootstrap/`）

- 全局状态设置
- MCP 客户端初始化
- 会话上下文恢复
- 工具和技能注册

## 12. Memdir 共享目录（`src/memdir/`）

智能体共享文件系统区域：
- 持久的跨工作者知识层
- 草稿板目录管理
- 路径通过协调者注入
