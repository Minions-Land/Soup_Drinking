## 1. 命令注册中心

中央注册表（`src/commands.ts`）从多个来源聚合命令：

| 来源 | 类型 | 示例 |
|------|------|------|
| 内置 | 本地/UI 命令 | `/exit`、`/help`、`/clear` |
| 技能 | 基于 Prompt 的 AI 命令 | `/commit`、`/review-pr` |
| 插件 | 动态加载 | 来自 `src/plugins/` |
| 工作流 | 可脚本化工具链 | 自定义工作流 |

### 命令类型

```typescript
type Command = {
  name: string
  type: 'prompt' | 'local' | 'local-jsx'
  // 'prompt' → 展开为 LLM 提示词
  // 'local'  → 执行 JS 逻辑，返回文本
  // 'local-jsx' → 渲染交互式 React/Ink UI 组件
}
```

### 常用斜杠命令

| 命令 | 用途 |
|------|------|
| `/help` | 显示帮助 |
| `/clear` | 清除对话 |
| `/compact` | 压缩对话历史 |
| `/config` | 编辑设置（交互式） |
| `/commit` | 创建 git 提交 |
| `/branch` | 切换/创建分支 |
| `/context` | 管理对话上下文 |
| `/agents` | 列出/管理智能体 |

### 过滤与分发

命令按以下条件过滤：
- **认证级别** — `claude-ai` vs `console` 用户
- **运行模式** — `REMOTE_SAFE_COMMANDS` 用于远程会话
- **Bridge 安全** — 适合移动端/网页端远程控制

## 2. CLI 基础设施

### 结构化 IO（`structuredIO.ts`）
基于 NDJSON 消息交换的基础层：
- 处理 `control_request`（权限提示）
- 处理 `control_response`（用户应答）

### 远程 IO（`remoteIO.ts`）
扩展结构化 IO 用于双向流式传输：
- 集成 CCR（Claude 控制 REPL）
- 心跳管理、Epoch 追踪、会话状态报告

### 传输层（`src/cli/transports/`）

| 传输 | 协议 | 用途 |
|------|------|------|
| `SSETransport` | Server-Sent Events | CCR v2 |
| `WebSocketTransport` | WebSocket | 全双工 |
| `HybridTransport` | WS+POST | 读=WS，写=POST |

## 3. Vim 模式（`src/vim/`）

终端输入的完整 Vim 模拟：

### 状态机

```
idle（空闲）→ count（计数）→ operator（操作符）→ operatorFind/operatorTextObj → idle
```

### 动作（`motions.ts`）
- `h`、`j`、`k`、`l` — 基本移动
- `w`、`b`、`e` — 单词移动
- `G`、`gg` — 行跳转
- `f`、`F`、`t`、`T` — 查找字符

### 操作符（`operators.ts`）
- `d` — 删除、`c` — 修改、`y` — 复制、`r` — 替换、`>`/`<` — 缩进

### 文本对象（`textObjects.ts`）
- `iw`/`aw` — 内部/周围单词
- `i"`/`a"` — 内部/周围引号

## 4. 插件系统（`src/plugins/`）

插件可提供：
- **技能** — 基于 Prompt 的命令
- **Hook** — 如 `PermissionRequest` 拦截器
- **MCP 服务器** — 外部工具提供者

内置插件可通过 `/plugin` UI 切换启用/禁用。

## 5. 快捷键系统（`src/keybindings/`）

- **多键组合** — 如 `ctrl+k ctrl+s`
- **上下文感知** — 作用域为 `Global`、`Chat`、`Vim`
- **可自定义** — 用户覆盖从设置加载

## 6. 迁移脚本（`src/migrations/`）

版本间设置自动迁移，如模型升级、上下文窗口升级等。

## 7. Buddy 伴侣功能（`src/buddy/`）

- **确定性生成** — 基于用户 ID 的种子 PRNG（`mulberry32`）
- **属性** — 物种、稀有度（普通→传说）、随机化属性
- **ASCII 精灵** — 通过 Ink 组件渲染
